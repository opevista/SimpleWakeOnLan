# SimpleWakeOnLan for iOS

**SimpleWakeOnLan for iOS** は、SwiftUI で開発された **iPhone / iPad
向け** の Wake on LAN（WoL）ユーティリティです。\
複数のデバイスを登録・管理し、同一ネットワーク上から Magic Packet
を送信してデバイスを起動できます。\
また、Ping
によるオンライン状態の確認や通信ログの記録にも対応しています。

※ 本アプリは **Wi-Fi 接続時のみ動作**します（キャリア回線・VPN・Private
Relay 非対応）

------------------------------------------------------------------------

## 特徴

-   複数デバイスの登録・編集・削除\
-   Wake on LAN（Magic Packet）送信\
-   Ping によるデバイスのオンライン／オフライン判定\
-   通信ログの自動記録\
-   SwiftUI による iOS ネイティブ UI\
-   ダークモード／ライトモード対応\
-   デバイス情報の自動保存（UserDefaults）

------------------------------------------------------------------------

## 動作環境

-   iOS 17.0 以降\
-   iPhone / iPad\
-   Xcode 15 以降（ソースビルド時）\
-   Swift 5.9 以降

※ 「My Mac (Designed for iPad)」では動作しません

------------------------------------------------------------------------

## 使用技術・ライブラリ

-   Swift / SwiftUI\
-   Network.framework（UDP 通信）\
-   SwiftyPing（Ping 実装）

------------------------------------------------------------------------

## インストール

### ソースコードからビルド

``` bash
git clone https://github.com/opevista/SimpleWakeOnLan.git
cd SimpleWakeOnLan
open SimpleWakeOnLan.xcodeproj
```

Xcode で **iOS ターゲット**を選択し、\
実機（iPhone / iPad）にビルド・実行してください。

------------------------------------------------------------------------

## 事前設定（重要）

初回起動時に表示される\
**ローカルネットワークへのアクセス許可** を必ず有効にしてください。

------------------------------------------------------------------------

## 使い方

### デバイスの追加

1.  右上の **＋** ボタンをタップ\
2.  以下の情報を入力
    -   Name：デバイス名\
    -   IP Address：対象デバイスの IP アドレス\
    -   MAC Address：AA:BB:CC:DD:EE:FF 形式\
    -   Broadcast Address：例 192.168.1.255\
    -   WoL Port：通常は 9\
3.  Save をタップして登録

------------------------------------------------------------------------

### Wake on LAN の送信

-   デバイス詳細画面の **Wake** ボタンをタップすると Magic Packet
    が送信されます。

------------------------------------------------------------------------

### ステータス確認

-   Check をタップすると Ping を送信し、状態を確認します。

  表示   状態
  ------ ---------
  🟢     Online
  🔴     Offline
  ⚪     Unknown

------------------------------------------------------------------------

## 注意事項

-   同一 LAN 内でのみ使用可能\
-   キャリア回線・VPN・Private Relay 使用時は通信不可\
-   対象デバイス側で WoL 有効化が必要
